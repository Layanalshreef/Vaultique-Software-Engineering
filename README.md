# Vaultique

## Overview
Vaultique is a Software Engineering project that introduces a digital platform for preserving memories by connecting physical keepsakes with digital content. The system organizes memories into structured life chapters using QR code integration.

## Key Features
- Memory Chapters
- QR Code Integration
- Digital & Physical Archive
- Collaborative Memories
- Interactive UI Design
- Secure Authentication
- UML Design
- Software Architecture
- Test Cases & Documentation

## Project Scope
This project includes:
- Requirements Analysis
- User Analysis
- Risk Assessment
- Software Architecture
- UI Wireframes & Mockups
- UML Diagrams
- Testing Documentation
